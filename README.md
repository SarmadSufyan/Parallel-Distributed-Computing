**Status:** Completed  
**Folder:** [`Task-01-Multiprocessing-vs-Multithreading/`](./Task-01-Multiprocessing-vs-Multithreading/)

**Description:**  
Comparative analysis of Python's multiprocessing and multithreading approaches for CPU-intensive tasks. Implements prime number calculation and mathematical operations to demonstrate the impact of Python's Global Interpreter Lock (GIL) on parallel performance.

**Key Concepts:**
- Multiprocessing vs Multithreading
- CPU-bound task optimization
- GIL impact analysis
- Performance benchmarking
- Process and thread management

**Technologies:** Python, multiprocessing, threading, time
